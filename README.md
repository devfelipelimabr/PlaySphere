# PlaySphere - Template de Site de Vendas de Jogos em Node.js

![PlaySphere Logo](https://github.com/devfelipelimabr/PlaySphere/blob/master/public/images/games-icon.png?raw=true)

Este repositório contém o código-fonte e os recursos necessários para o template do site de vendas de jogos PlaySphere em Node.js. Com este template, você pode criar seu próprio site de vendas de jogos online, com funcionalidades como exibição de jogos, carrinho de compras, gerenciamento de usuários, processamento de pagamentos e muito mais.

## Características

- Design responsivo e atraente para uma experiência de usuário moderna.
- Página inicial com destaques e jogos em destaque.
- Catálogo de jogos com pesquisa e filtragem.
- Páginas individuais para cada jogo, com informações detalhadas e imagens.
- Carrinho de compras para adicionar e gerenciar os jogos selecionados.
- Funcionalidade de pagamento para finalizar as compras.
- Área do usuário para gerenciar o perfil, histórico de pedidos e detalhes da conta.

## Pré-requisitos

Certifique-se de ter os seguintes softwares instalados em sua máquina:

- Node.js
- NPM (gerenciador de pacotes do Node.js)

## Instalação

1. Clone este repositório em sua máquina local.
2. No diretório raiz do projeto, execute o seguinte comando para instalar as dependências:

```
npm install
```

3. Após a instalação das dependências, execute o seguinte comando para iniciar o servidor local:

```
node index.js
```

4. Abra um navegador da web e acesse `http://localhost:45678` para ver o site em execução.

## Personalização

Você pode personalizar o template de acordo com suas necessidades, alterando o design, adicionando novas funcionalidades ou integrando serviços externos. Explore o código-fonte e os arquivos para entender a estrutura e a lógica por trás do template.
